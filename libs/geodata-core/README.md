# 🌍 Geolijn Geodata Core

@geolijn/geodata-core is een Angular library die zich richt op geodata-verwerking en analyse. De library biedt hulpmiddelen voor het werken met polygonen, lijnen en punten, waardoor je eenvoudig ruimtelijke berekeningen kunt uitvoeren binnen je Angular-projecten.

Deze library is ontworpen als kernmodule voor geografische toepassingen – ideaal voor projecten waar je geometrieën wilt analyseren, visualiseren of transformeren.

This library was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.0.
