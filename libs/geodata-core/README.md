# 🌍 Geolijn Geodata Core

@geolijn/geodata-core is een Angular library die zich richt op geodata-verwerking en analyse. De library biedt hulpmiddelen voor het werken met polygonen, lijnen en punten, waardoor je eenvoudig ruimtelijke berekeningen kunt uitvoeren binnen je Angular-projecten.

Deze library is ontworpen als kernmodule voor geografische toepassingen – ideaal voor projecten waar je geometrieën wilt analyseren, visualiseren of transformeren.

This library was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.3.0.

## ✨ Wat doet deze library?
De library bevat een set van geometrische analysetools die veelvoorkomende ruimtelijke bewerkingen eenvoudig maken. Denk hierbij aan:

Het berekenen van kenmerken van geometrieën zoals het centroid, de oppervlakte en de perimeter van een polygoon.

Het uitvoeren van afstandsbepalingen tussen punten, bijvoorbeeld met de Haversine-formule voor nauwkeurige afstanden op basis van geografische coördinaten.

Het genereren van afgeleide geometrieën, zoals een buffer (een polygoon uitbreiden met een bepaalde afstand) of het bepalen van de bounding box van een geometrie of collectie geometrieën.

Het analyseren van ruimtelijke relaties zoals intersecties of overlaps (bijvoorbeeld controleren of twee polygonen elkaar snijden).

Het vereenvoudigen van complexe geometrieën met algoritmes zoals Douglas-Peucker, zodat je lichtere en snellere geometrieën krijgt voor visualisatie of opslag.

Kortom: met deze library kun je ruimtelijke analyse en geodata-transformatie uitvoeren zonder externe GIS-software.

## ✅ Wanneer gebruik je deze library?
Als je polygonen, lijnen of punten wilt bewerken of analyseren in Angular.

Bij toepassingen waar je kaartvisualisaties hebt (bijvoorbeeld met Leaflet of OpenLayers) en extra ruimtelijke logica nodig is.

Voor het voorbereiden van geodata voor opslag, visualisatie of verdere analyse.
