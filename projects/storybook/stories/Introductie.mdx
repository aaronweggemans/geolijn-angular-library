<div>
  <h1>Geolijn Libraries</h1>

  <p>
    Binnen Geolijn zijn er meerdere Angular libraries beschikbaar die helpen bij het bouwen van geografische toepassingen.
    Deze libraries vullen elkaar aan en bieden zowel UI-componenten als geavanceerde geodata-functionaliteit.
  </p>

  <h2>@geolijn/geodata-core</h2>

  <p>
    <strong>@geolijn/geodata-core</strong> is een Angular library die zich richt op geodata-verwerking en analyse.
    De library biedt hulpmiddelen voor het werken met polygonen, lijnen en punten, waardoor je eenvoudig ruimtelijke
    berekeningen kunt uitvoeren binnen je Angular-projecten. Deze library is ontworpen als
    kernmodule voor geografische toepassingen – ideaal voor projecten waar je geometrieën wilt analyseren, visualiseren
    of transformeren. De library is beschikbaar als een npm package:
  </p>

  ```npm
    npm install @geolijn/geodata-core
  ```

  <h2>@geolijn/generieke-componenten</h2>
  <p>
    <strong>@geolijn/generieke-componenten</strong> is een Angular library die zich richt op herbruikbare componenten.
    De componenten kunnen gebruikt worden voor geografische oplossingen en bieden onder andere geo-specifieke UI-elementen zoals
    kaarten, een locatiezoeker en meer. De library is beschikbaar als een npm package:
  </p>

  ```npm
    npm install @geolijn/generieke-componenten
  ```
</div>
